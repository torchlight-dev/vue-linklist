===
## Installation
```
npm install --save vue-linklist
```
or
```
yarn add vue-linklist
```

## Usage
```js
import vueLinklist from 'vue-linklist';
```
or
```js
const vueLinklist = require('vue-linklist');
```

and see example!

## Launch example
```bash
$ cd example
$ yarn install    // or npm install
$ npm start
```

## How to build
```
$ npm run build
```
